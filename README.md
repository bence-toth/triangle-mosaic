# Triangles

## Options

### Coloring modes

#### Single color

```js
coloring: {
  mode: 'single',
  color: '#ffc107'
}
```

#### Spots

```js
coloring: {
  mode: 'spots',
  spots: [
    {
      x: 0,
      y: 0,
      color: '#ffc107'
    },
    {
      x: 1280,
      y: 0,
      color: '#f44336'
    },
    {
      x: 640,
      y: 720,
      color: '#2196f3'
    }
  ],
  spotStrength: 2.5,
}
```
